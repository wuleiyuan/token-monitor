# Token Monitor - 开发记忆

## 工作流程 (必须遵循)

```
GitHub 目录修改 → 验证 → 推送到 GitHub → 更新 DEVELOPMENT_LOG.md
```

### 流程说明

1. **GitHub 目录** (`/Users/leiyuanwu/GitHub/token-monitor/`) - 开发测试环境
2. **验证** - Python 语法检查 + 模块导入测试
3. **推送** - `git push origin main`
4. **更新** - 修改 DEVELOPMENT_LOG.md 记录本次开发内容

### 两个项目目录

| 目录 | 用途 |

| `/Users/leiyuanwu/GitHub/token-monitor/` | GitHub 同步目录 - 发布到 PyPI |

